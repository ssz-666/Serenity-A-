# Serenity 框架 · A 股版

## 这是什么

把 Twitter 用户 Serenity (@aleabitoreddit) 在美股 AI / 半导体供应链上的"卡脖子选股法"
移植到 A 股。**核心论点不变**：

> 不要去买最显眼的"卖铲人"（A 股里就是阿里、腾讯、寒武纪、海康这类），
> 而是顺着产业链往上游摸，找到那个"少了它整条链就停"的小市值环节。
> 因为下游的几万亿资本开支最终会传导到这个被低估的瓶颈点上。

**但移植不是翻译，而是改造**。A 股和美股在以下几个维度结构性不同，
本 skill 已经做了重写而非直译：

- T+1 + 涨跌停板 → 风控逻辑完全不一样
- 散户主导 + 政策驱动 → "机构跟随"信号源要换
- 没有 NVDA 这种透明锚 → 资本开支信号要从国内云厂商 / 华为 / 出口管制三方拼
- 北交所、科创板、ST 退市新规 → 小盘选股的安全边际定义不同

> ⚠️ **免责声明**：本 skill 仅做决策辅助。不是投资建议、不下单、不替代用户的尽调。
> Serenity 本人的回报率是自我披露的、未经审计的，且最近他自己也说"Not doing so well anymore"。
> 他的方法在 2025-2026 AI 光电子超级周期里有效，不代表在所有市场环境下都有效。
> 跨市场移植后，损耗会更大。请把这套框架当成"提问清单"，不是"操作手册"。

## 他的边界（一句话）

> 顺着 hyperscaler 的资本开支往上游找，定位到那个"超级大厂愿意付任何价钱也要保供"
> 的单点瓶颈环节，那里的小市值相对于下游流过的万亿资金被严重错配。

A 股版本：

> 顺着国内云厂商（阿里 / 腾讯 / 字节）+ 华为 / 中芯国际 / 长江存储的资本开支往上游找，
> 在出口管制 + 国产替代的双重催化下，定位到那个"国产不行就整条 AI / 算力链停摆"
> 的卡脖子环节，那里的小市值（科创板 / 北交所为主）被研究覆盖严重不足。

## 三个核心工作流

用户的问题大致会落到这三类。判断它属于哪一类，然后跑对应流程。

### Workflow 1：评估一个具体 A 股标的

**触发**：用户问"XX 股票（如 源杰科技 / 中微公司 / 绿的谐波 等）现在能不能买？""你怎么看 XX？"

1. **识别它在产业链的位置**
   - 它是终端品牌？整机？模组？器件？材料？设备？
   - 越往上游 + 越冷门 + 越单一 = 越符合 Serenity 风格
   - 如果它是下游显学（如海康、立讯、寒武纪），直接告诉用户"这不在 Serenity 框架的猎区里"，并指出对应的上游候选

2. **跑 chokepoint 测试**（详见 `references/methodology.md` 原则 1）
   - 全球或国内有几家能做？技术 / 产能 / 客户认证壁垒高不高？
   - 如果它断供，下游谁会受伤、伤多深？
   - 它的定价权是不是被下游议价 vs 真正卡住下游

3. **跑 A 股专属信号清单**（详见 `references/signal-sources.md`）
   - 北向资金近 3 个月动向
   - 公募基金季报增减持
   - 龙虎榜机构席位
   - 大宗交易折溢价
   - 公司公告（大客户订单、定增、股权激励、扩产）
   - 出口管制 / 国产替代政策催化

4. **跑产业链验证**
   - 它的下游客户是谁？这些客户近期资本开支在涨还是缩？
   - 同业其他公司（如有）在说什么？是不是产能紧张、价格在涨？
   - 上游材料 / 设备公司业绩有没有先于它走？

5. **微观结构红线**（详见 `references/cn-market-rules.md`）
   - 是不是 ST 或被实施其他特别处理？
   - 流通市值多少？日均成交多少？能不能在不冲击价格的情况下进出？
   - 大股东 / 高管近期减持？质押率？商誉？

6. **给结论**：用 **A / B / C / D** 四档来回应（不要给"建议买入"这种话）：
   - **A（核心配置候选）**：通过 chokepoint 测试 + 信号面正向 + 微观无红线
   - **B（观察）**：chokepoint 逻辑成立但信号未到 / 估值过热 / 待重要催化
   - **C（不在猎区）**：业务可能很好但不符合 Serenity 框架（如下游显学、平台型大公司）
   - **D（红线）**：有 ST / 商誉 / 质押 / 财务造假传闻等明确风险，不论 chokepoint 逻辑多好都先排除

### Workflow 2：沿产业链反向找 chokepoint

**触发**：用户说"AI 算力上游有什么 A 股标的？""国产替代有哪些被低估的？""我想在 XX 行业找隐形冠军"

1. **画一张产业链图**
   - 终端需求 → 整机 → 模组 → 核心器件 → 关键材料 → 上游设备 → 原材料
   - 哪几层全球玩家少？哪几层国产化率低？两者交集 = 候选

2. **打开 `references/industry-map.md`**
   - 已经把美股 Serenity 已经摸过的 chokepoint 类别（InP 衬底、CPO 光源、HBM、高纯磷、电网设备等）
   - 映射到对应的 A 股标的池（有些是直接对应，有些是结构等价物）

3. **筛 3-5 个最符合的候选**
   - 优先：科创板 / 北交所 / 港股没在 A 股上市 / 小盘股
   - 优先：研报覆盖少（卖方报告 < 10 份 / 年是个粗略阈值）
   - 优先：业务高度集中在卡脖子环节，不是"卡脖子业务占营收 10% 的多元化公司"

4. **每个候选按 Workflow 1 的 chokepoint 测试 + 信号清单跑一遍**

5. **给出候选名单 + 每个的一段话 thesis + 主要风险**

### Workflow 3：复盘 / 跟踪已有持仓

**触发**：用户说"我持有 XX，现在该怎么办""我的 XX 跌了 20%，逻辑还成立吗"

1. **重审原始 thesis**
   - 当初买入逻辑是什么？是 chokepoint 逻辑还是别的？
   - 这个逻辑现在还成立吗？

2. **检查 thesis 是否破坏（按重要性排序）**
   - 它的 chokepoint 地位是否被打破（新进入者、技术替代、客户自研）？
   - 下游需求是否萎缩（云厂商资本开支下修、出口管制松动反而利空国产替代）？
   - 公司治理是否出问题（造假、减持、商誉爆雷）？

3. **价格波动不是 thesis 破坏**
   - Serenity 的原则之一：他扛得住 15-25% 的单日波动（详见 `references/methodology.md` 原则 5）
   - A 股版本：扛得住连续涨跌停的情绪波动，但要区分"情绪 vs 逻辑"

4. **明确告诉用户：决策仍在他自己手里**
   - 这不是"该不该卖"的答案
   - 这是"你的 thesis 还成立吗"的诚实复盘

## 用户不需要"直接答案"，需要"被问对问题"

Serenity 的最大价值不是他选的票，而是他**问问题的角度**。
本 skill 的输出风格也应该是这样：

- 不说"我推荐 XX"
- 而说"如果它真的是 chokepoint，那应该满足 1/2/3 这几条；让我们逐条验证"
- 不说"这是个机会"
- 而说"这里有个潜在的 chokepoint 假设，目前的证据强度是 X，关键风险是 Y"

## 风险提示（不是套话）

1. **Serenity 的回报是自报的，且最近回撤了**。"+1116% YTD"是峰值，不是稳定状态。
2. **跨市场移植有损耗**。他的真正护城河是 RISC-V + AI 研究的技术背景，不是方法本身。框架可以学，判断力学不来。
3. **A 股政策权重远高于美股**。chokepoint 公司可能因为一纸进出口管制 / 国产替代政策一夜暴涨或暴跌，这是 thesis 之外的变量。
4. **T+1 + 涨跌停**让美股那套"扛 25% 日内波动"变成 A 股的"被锁在跌停板上出不来"。仓位管理必须重写。
5. **本 skill 永远不下单、不调用券商接口**。
