# A 股市场微观结构（与美股的关键差异）

> 这部分是 A 股版 skill 相对原版的"硬约束"补充。
> Serenity 在美股那套"扛 25% 日内波动 + 看到回撤就加仓"在 A 股要重写，
> 因为基础规则不一样。

## 1. T+1 + 涨跌停板（最大的结构性差异）

**规则**：
- 当天买入的股票，当天不能卖（T+1）
- 主板 / 中小板：单日涨跌幅 ±10%
- 科创板 / 创业板：单日涨跌幅 ±20%
- 北交所：单日涨跌幅 ±30%
- 新股上市前 5 个交易日不设涨跌停

**对 Serenity 框架的影响**：
- 美股"日内 25% 回撤后加仓"在 A 股变成"跌停后第二天才能再操作"
- 一旦标的连续跌停，**你被锁在里面**，无法平仓——这是美股完全没有的风险形态
- 高位连续涨停的股票，看似强势，但一旦反转可能连续跌停（如 2024 年多个 AI 概念股的"天地板"行情）

**风控规则**：
- 单一标的仓位上限：建议不超过总仓位 15%（美股 Serenity 可能 25-40% 单一仓位，A 股别学）
- 总仓位 ≤ 70%，**永远留 30% 现金应对连续跌停下的加仓机会或保命**
- 不要在涨停板上追入（涨停后第二天高开低走是常态）

## 2. ST 制度 + 退市新规

**规则**：
- 连续两年净利润为负 → \*ST
- 连续三年净利润为负 + 营收 < 1 亿 → 退市风险警示
- 财务造假 / 信息披露重大违法 → 强制退市
- 新规下退市速度比以前快得多

**对 Serenity 框架的影响**：
- Serenity 喜欢的"小市值 + 低覆盖" 在 A 股有**变成 ST 的尾部风险**
- chokepoint 逻辑再强，公司被 ST / 退市，你的钱归零

**风控规则**：
- 看 chokepoint 之前先看**财务健康度**：
  - 净利润：连续两年正
  - 经营性现金流：与净利润匹配（避免应收账款 / 商誉造假）
  - 资产负债率：不超过同行均值 + 20%
  - 商誉 / 净资产：< 30%
- 大股东质押率 > 60% 是黄灯，> 80% 是红灯（爆仓风险）
- 公司或大股东被立案调查 / 实控人变更频繁 = 直接排除

## 3. 政策催化的双刃剑

**美股**：政策催化偶发，影响相对可预测
**A 股**：政策是定价的核心变量之一，远超基本面在某些阶段的权重

**典型政策催化事件（chokepoint 标的的常见 driver）**：
- 出口管制升级（如美国 BIS 实体清单） → 国产替代板块跳涨
- 国家大基金 / 地方产业基金投资 → 直接利好
- "新质生产力 / 国产替代 / 自主可控" 类政策表态 → 板块情绪驱动
- 国务院 / 工信部 / 发改委专项规划发布 → 板块持续催化

**对 Serenity 框架的影响**：
- 一个 chokepoint 公司可能因为政策一夜暴涨 50%，也可能因为政策松动一夜跌 20%
- 政策的"长尾"效应在 A 股很重要：第一次催化往往是龙头吃肉，第二次轮动才到二线

**风控规则**：
- 不要在政策催化高潮买入（典型如"某概念出台当天涨停后追入"）
- 政策证伪信号要敏感（如海外管制松动 → 国产替代逻辑承压）
- 长期 thesis 不依赖单一政策延续

## 4. 信息披露的特点

**美股**：13F / 13D / 13G 强制披露大资金动向
**A 股**：

| 披露类型 | 频率 | 价值 |
|---------|------|------|
| 5% 以上股东持股变动 | 触发披露 | 高 |
| 高管 / 监事增减持 | 触发披露 | 高（高管买入是强信号）|
| 公募基金前十大持仓 | 季度 | 中（有滞后）|
| 北向资金 | 每日 | 高 |
| 龙虎榜机构席位 | 每日（限触发条件） | 中高 |
| 大宗交易 | 每日 | 中 |
| 招股说明书 / 年报 / 定期报告 | 法定周期 | 高 |
| 股东户数 | 半年 / 季度 | 中（户数减少 = 筹码集中）|

**对 Serenity 框架的影响**：
- A 股的"机构跟随确认"比美股更分散，需要看多个数据源拼图
- 详见 `signal-sources.md`

## 5. 流动性差异

**美股**：纳斯达克 / NYSE 流动性极好，5 亿美元市值的股票也能正常交易
**A 股**：流动性分层明显

| 板块 | 单日成交（活跃股 / 不活跃股）|
|------|----------------------|
| 主板大盘 | 数亿至数十亿 |
| 科创板 / 创业板小盘 | 数千万至数亿 |
| 北交所 | 几百万至几千万 |

**对仓位的影响**：
- 自有资金 100 万：可以做几乎所有标的
- 自有资金 500 万：北交所要小心进出冲击成本
- 自有资金 1000 万+：科创板 / 北交所小盘要分批建仓 / 出货

**风控规则**：
- 进出场都用"不超过当日成交量 5%"的硬约束
- 北交所建仓周期通常需要数日至数周

## 6. 融资融券 + 雪球 + DMA 的劝退

**美股 Serenity 原则 7：不用期权 / 不用杠杆**
**A 股版本**：

| 工具 | 为什么不建议用 |
|------|--------------|
| 融资买入 | T+1 + 涨跌停 = 反向后无法当日平仓，强平连环风险 |
| 雪球结构 | 敲入风险在 A 股高波动下经常触发 |
| 个股期权 | 高门槛、低流动性、theta 损耗大 |
| 私募 DMA / 收益互换 | 散户接触不到，也不建议 |
| 股指期货对冲 | 适合大资金，散户没必要 |

**铁律**：跑 Serenity 框架，**永远用现金仓位**。

## 7. 新股 / 次新股的特殊性

**美股**：IPO 后流动性 / 估值波动正常
**A 股新股**：
- 上市初期可能涨幅惊人（破发也有，但远少于成熟市场）
- 6 个月 / 12 个月解禁期是关键节点
- 战略配售 / 网下配售解禁会带来抛压

**对 Serenity 框架的影响**：
- 一个完美符合 chokepoint 定义的新股，可能因为解禁压力被压制 6-12 个月
- 关注上市公司的 **限售股解禁时间表**，避开大额解禁前一个月

## 总结：把美股原则翻译成 A 股语言

| 美股原则 | A 股版本 |
|---------|---------|
| 扛 25% 日内波动 | 扛连续 2-3 个跌停板的情绪冲击 |
| 看到大跌就加仓 | 看到大跌**第二天**判断是否加仓；连续跌停时停止动作等流动性恢复 |
| 单一仓位 25-40% | 单一仓位 ≤ 15%，留 30% 现金 |
| 跟踪 13F / 机构持仓 | 拼图：北向 + 公募 + 龙虎榜 + 大宗 + 股东户数 |
| 政策影响有限 | 政策是定价核心变量之一 |
| 看财报基本面 | 看财报 + ST / 商誉 / 质押 / 解禁 / 实控人变更 |
| 不用期权 | 完全不用杠杆（融资 / 雪球 / DMA / 个股期权 / 配资） |
