# Methodology · Serenity 选股 8 条原则（A 股改写版）

> 这是 Serenity 在公开推文里反复展现的可迁移原则。
> 每条都按 **原始逻辑 → A 股改写 → 怎么用** 的结构重写。
> 原文出处：综合 yan-labs/serenity-aleabitoreddit、leslieyeo/serenity-reply
> 以及 singularityresearchfund.substack.com 的第三方深度分析。

---

## 原则 1：Chokepoint 测试（卡脖子测试）

**原始逻辑**：不投显眼的卖铲人（NVDA、META），投他们供应链上游那个"少了就停"的单点环节。
判断 chokepoint 的三个问题：
1. 全球能做这件事的公司是不是 ≤ 3 家？
2. 替换它需要多长时间？（理想：> 2 年）
3. 它的下游客户愿意接受涨价吗？

**A 股改写**：在国内做 chokepoint 分析有两个 catalyst 维度：
- **全球性 chokepoint**：在全球供应链里独家或寡头（如三安光电的 InP 衬底、源杰科技的 DFB 激光芯片）
- **国产替代 chokepoint**：技术上不一定全球领先，但在"中国必须自主可控"的政策语境下，是国内唯一/少数选项（如中微公司的等离子刻蚀机、华海清科的 CMP 设备）

第二种是 A 股独有的机会类型，**美股没有对应物**，但更易受政策反复影响。

**怎么用**：拿到一个标的，逐条问：
- 它的核心业务有几家全球或国内竞争者？说出名字
- 客户的替换成本是多少？（认证周期 / 重新设计成本 / 沉没投资）
- 最近一次出现过下游被迫接受涨价的事件吗？
- 这个 chokepoint 是技术性的（壁垒持久）还是政策性的（壁垒可变）？

**红旗**：如果一个公司说自己"国产替代受益"但全球已经有 5+ 家成熟玩家，
说明它只是个跟随者，不是 chokepoint。

## 原则 2：跟踪资本开支龙头的信号（"NVIDIA 跟随"的 A 股版本）

**原始逻辑**：NVDA 的资本开支去向 / 投资动作 / 客户认证 = 整个 AI 供应链最早的领先信号。
比如 NVDA 投了 Marvell 2B 美元做 silicon photonics → 验证 CPO 路线 → 利好 $SIVE 等上游。

**A 股改写**：A 股没有 NVDA 这种"信号集中器"，需要把信号源拆开多路追踪：
1. **国内云厂商资本开支**：阿里巴巴 / 腾讯 / 百度 / 字节跳动的财报披露 + 招标公告
2. **华为的供应链动作**：华为没上市，但供应链订单 leak、官方发布会、Mate / Pura 系列发布 = 强信号
3. **中芯国际 / 长江存储 / 长鑫存储扩产**：晶圆厂扩产 = 上游设备 / 材料 / 化学品需求
4. **美国出口管制清单变化**：每次扩大管制 → 国产替代名单内公司订单跳涨
5. **国家大基金 / 地方产业基金的投资动向**：大基金一期 / 二期 / 三期投了谁，是非常强的信号

**怎么用**：每次分析一个标的，问自己：
- 它最大的下游客户最近一次资本开支或扩产公告是什么？
- 它在不在某次出口管制名单的"受益替代"清单上？
- 大基金或地方产业基金最近投了它或它的同业吗？

**关键区别**：美股的信号是 hyperscaler 财报里写出来的；
A 股的信号更多在政策文件、招标网站、产业链调研纪要里。
这意味着工作量更大，但也意味着信息不对称的窗口更长。

## 原则 3：研究覆盖稀薄优先（"欧洲小盘"的 A 股版本）

**原始逻辑**：Serenity 偏爱欧洲小盘，因为美国机构覆盖少，定价更低效。
比如 $LPK（德国玻璃基板）、欧洲半导体设备小厂。

**A 股改写**：A 股的等价物有三个：
1. **北交所**：整体研究覆盖最稀薄的板块
2. **科创板小盘**：研报 < 10 份 / 年的二三线标的
3. **港股小盘 + 没有 A 股 ADR 的 H 股**：被两边都遗忘的中间地带

**怎么用**：粗筛标准
- 卖方研报数量：年度 < 15 份是低覆盖
- 公募基金持仓家数：< 30 家是机构关注度低
- 通过 Wind / 同花顺 / 东方财富的"研报"页面查看

**注意**：低覆盖 ≠ 低估值。
有的小盘股是因为有问题（治理、财务、商誉）所以没人看。
chokepoint 测试 + 微观结构红线（见 `cn-market-rules.md`）必须先过。

## 原则 4：拒绝"妖股 / 题材股"标签的污名（反 meme stock 标签）

**原始逻辑**：Serenity 推过的 $AXTI 早期被骂"meme stock"，他的回应是：
"被骂 meme stock 的我的票后来都成了 billion-dollar 公司。"
意思是：散户追捧 ≠ 没基本面，市场情绪标签不能替代逻辑分析。

**A 股改写**：A 股版本的污名是"妖股 / 炒概念 / 题材股"。
但要区分两种：
- **真题材**：政策催化 + chokepoint 逻辑 + 业绩兑现路径（如 2023 年的 CPO / 光模块行情，本质是真供给侧紧张）
- **假题材**：纯情绪 + 蹭概念，业绩兑现路径不清楚（如部分蹭"算力"概念但实际在做卖咖啡 / 做地产的公司）

**怎么用**：当一个公司被骂"妖股"时，逐条检查：
- 主营业务的卡脖子逻辑是否扎实？
- 业绩兑现路径（订单 / 产能 / 客户）是否清晰？
- 涨幅主要来自基本面变化还是纯情绪传染？

如果前两个答案都是"是"，那这个"妖股"标签可以忽略。
如果有一个答案是"否"，远离。

## 原则 5：扛住短期波动，但区分"情绪波动 vs 逻辑破坏"

**原始逻辑**：Serenity 持有标的能扛 15-25% 的日内波动，对应他的高确信度。
但他不是"死扛"——他对 $IREN 的态度反转就显示了他会在 thesis 变化时调整。

**A 股改写**：A 股的波动结构和美股很不一样：
- 没有 25% 日内波动（涨跌停板锁死在 10% / 20% / 30%）
- 但有**连续涨跌停**和**复牌后补跌**这种美股没有的形态
- 也有**ST 戴帽**等突发风险

判断要不要扛的核心问题：
- 波动来自情绪传染（如行业杀估值、北向流出、风格切换）→ 可以扛
- 波动来自基本面破坏（客户流失、技术替代、政策反转、财务问题）→ 必须重审 thesis

**怎么用**：每次出现 ≥ 10% 单日下跌或连续 3 日下跌时，问：
1. 跌幅来自公司基本面变化还是市场整体？
2. 同行业 / 同 thesis 的其他标的也在跌吗？（如果是，更可能是行业 / 情绪问题）
3. 公司最近有没有出新公告 / 业绩预告？
4. 我的原始 thesis 有没有任何一条被打破？

如果四个问题都指向"情绪"，那扛住。
如果有一个指向"基本面"，重新评估。

## 原则 6：机构跟随确认（institutional follow-on）

**原始逻辑**：Serenity 经常在公开推文里指出"Point72 / 大摩 / 高盛已经开始建仓 / 出研报"
作为 thesis 被机构验证的信号。

**A 股改写**：A 股没有 13F 的强制披露，但有更丰富的信号源：
1. **北向资金**：每日可查，方向性强
2. **公募基金季报**：每季度披露前十大持仓
3. **龙虎榜机构席位**：每天披露，但只在涨跌幅触发条件时
4. **大宗交易**：每日可查溢折价 + 接盘方
5. **重要股东持股变动公告**：5% 以上股东必须披露
6. **股东户数变化**：户数减少 = 筹码集中 = 通常机构在买

详见 `signal-sources.md`。

**怎么用**：当一个 chokepoint 候选标的开始出现：
- 北向资金连续净流入 ≥ 2 周
- 同时公募基金家数环比增加
- 同时龙虎榜上出现机构席位买入

= "机构跟随确认"的 A 股版本。**不是入场理由，但是 thesis 没瞎选**的验证。

## 原则 7：反期权 / 反杠杆铁律

**原始逻辑**：Serenity 多次明确表示**只持股票本体**，不碰期权。
即使他的 thesis 对，期权也会因为 theta / vega 让你输钱。

**A 股改写**：A 股散户的杠杆工具是：
- 融资融券
- 雪球结构产品
- 个股期权（场内 50ETF / 300ETF，门槛较高）
- 私募的 DMA / 收益互换
- 配资（很多是违规的）

**铁律的 A 股版本**：
- chokepoint 思路本来就是"等机构发现 + 估值重估"，需要时间，不适合杠杆
- 杠杆 + T+1 + 涨跌停 = 一旦反向，没有日内平仓的退出窗口
- 即使 thesis 完全对，融资爆仓让你看不到回报

**怎么用**：永远用现金仓位。如果用户问"能不能上融资"，
回答永远是"按 Serenity 框架，不可以——不是因为你 thesis 错，是因为
即使 thesis 对你也可能在等待期被强平。"

## 原则 8：DYOR 底线（Do Your Own Research）

**原始逻辑**：Serenity 在被大量粉丝 copytrade 后，反复强调"我分享我的研究，不是替你决策"。
他公开承认"我也会错，最近 not doing so well"。

**A 股改写**：这条**完全平移**，但要更重。原因：
- A 股散户跟风文化更重，"抄作业"是常态
- A 股博主 / 大 V 推荐的法律 / 道德边界比美股更模糊
- 信息不对称 + 政策风险让"抄作业"的尾部风险更大

**怎么用**：每个 Workflow 的最后一步都要明确：
- 这是框架分析，不是推荐
- 用户必须自己做最后一步尽调（看招股书 / 看年报 / 看产业链调研纪要 / 看政策文件）
- 仓位决策永远在用户自己

如果用户问"那我到底买不买"，回答永远是：
"你的 thesis 现在的证据强度是 X，主要风险是 Y，决策权在你。"
**不替用户下决定，是这个 skill 的核心约束之一。**
