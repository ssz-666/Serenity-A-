# Signal Sources · A 股 Chokepoint 信号源拼图

> Serenity 在美股盯一个 NVDA 就能拿到大半信号。
> A 股没这个 luxury。需要从 6-8 个数据源拼图。
> 这份文件按"信号源 → 怎么查 → 怎么解读"的格式列出。

## 1. 国内云厂商资本开支

**为什么重要**：他们就是 A 股版的 hyperscaler。资本开支决定算力链全部上游需求。

**追踪对象**：
- 阿里巴巴（港股：9988、美股：BABA）
- 腾讯（港股：0700）
- 百度（港股：9888、美股：BIDU）
- 字节跳动（未上市，靠新闻 / 行业报告 / 招标网站间接追踪）
- 美团（港股：3690）
- 京东（港股：9618、美股：JD）

**查什么**：
- 季度财报里的 "Capital Expenditure" 项
- 业绩会上对资本开支指引的表态
- 大额采购公告（如阿里宣布未来 3 年投 3800 亿做 AI 基建）

**解读**：
- 资本开支同比 +30% 以上 → 上游强信号
- 资本开支同比下滑 → 警惕上游存货积压 / 价格回调
- 业绩会上"AI 基建投入加码"的表述 → 板块情绪催化

## 2. 华为 / 中芯国际 / 长江存储 / 长鑫存储动作

**为什么重要**：这些是国产替代主线的"事件中心"。

**追踪对象**：
- 华为：每年秋季发布会、运营商 BG 招标、Mate / Pura 系列拆机分析
- 中芯国际（港股：0981、A 股：688981）：季度财报、扩产公告
- 长江存储 / 长鑫存储：未上市，看招标网站 + 设备厂订单披露

**关键信号**：
- 华为新品供应链 leak（如 Mate60 国产化率突破 90%）
- 中芯国际宣布新厂投产（直接利好设备 / 材料链）
- 长江存储宣布扩产（HBM / NAND 国产化加速）

**怎么查**：
- 公司公告 + 同花顺 / Wind 公告聚合
- 行业自媒体：芯智讯、爱集微、半导体行业观察
- 招标网站：中招联合招标网、华为电子招标网

## 3. 美国出口管制变化

**为什么重要**：每次美国 BIS 加码管制 → 国产替代板块的爆发性催化

**追踪对象**：
- 美国商务部 BIS（Bureau of Industry and Security）公告
- "实体清单"（Entity List）扩展
- "外国直接产品规则"（FDPR）调整

**解读 framework**：
- 新增管制品类 → 该品类的 A 股国产替代标的爆发
  - 例：高端 AI 芯片管制 → 国产 GPU 主题
  - 例：先进刻蚀机管制 → 中微公司主题
- 实体清单新增中国公司 → 该公司的国内供应商受益
- 管制松动 → 国产替代逻辑承压（小心）

**怎么查**：
- 美国商务部官网（bis.gov）
- 中文媒体：财联社、第一财经的"管制"专题

## 4. 国家大基金 + 地方产业基金动向

**为什么重要**：国家级资本的方向就是政策方向。

**追踪对象**：
- 国家集成电路产业投资基金（"大基金"）：一期 / 二期 / 三期
- 各省市集成电路产业基金（上海、合肥、武汉、深圳）
- 国家科技成果转化基金
- 中央汇金、国新投资

**关键信号**：
- 大基金二期 / 三期投资公告
- 地方政府引导基金进入某细分领域
- 国新投资 / 汇金增持某些科技股

**怎么查**：
- 大基金投资标的：天眼查、企查查搜"国家集成电路产业投资基金"
- 上市公司公告"获得国家大基金 X 期入股"

## 5. 北向资金

**为什么重要**：A 股最及时的"机构方向"指标。

**怎么查**：
- 同花顺 / 东方财富 / Wind 都有北向资金每日板块流入榜
- 个股北向持仓数据：Choice 数据、Wind、东方财富

**解读规则**：
- 连续 2 周以上净流入 + 增持比例 > 持股 5% → 强信号
- 单日大额流入易反复，不作为信号
- 关注"北向资金加仓但股价没涨" → 筹码沉淀，后续可能爆发

**注意**：北向资金本身有"被动跟随指数"成分，要区分主动选股 vs 指数权重调整。

## 6. 公募基金持仓变化

**为什么重要**：A 股机构最大主体。季报披露的前十大持仓是关键拼图。

**怎么查**：
- 季报披露时间：1 月底（Q4）、4 月底（Q1）、7 月底（Q2）、10 月底（Q3）
- 数据源：东方财富 Choice / Wind / 同花顺 iFinD

**解读规则**：
- 单一基金加仓不代表什么；**持仓基金家数环比增加**才是信号
- 知名"灵魂基金经理"（如某些深耕科技的）首次进入 → 注意
- 全行业基金家数下降 + 流通市值在跌 → 警惕 thesis 出问题

**滞后性**：公募季报披露有 1 个月滞后，信号已半失效，配合北向使用。

## 7. 龙虎榜（机构席位 + 营业部）

**为什么重要**：A 股每日异动股的"现场调查"。

**怎么查**：
- 同花顺 / 东方财富每日"龙虎榜"页面
- 触发条件：日涨跌幅偏离值 ≥ 7%、振幅 ≥ 15%、换手率 ≥ 20% 等

**解读规则**：
- "机构专用席位"买入 = 机构主动建仓信号（强）
- 知名游资营业部（"作手新一"、"宁波桑田路"、"上塘路"等）买入 = 短期情绪信号（中）
- 北向资金（"中信证券深圳分公司"、"香港中央结算"）通过龙虎榜暴露 → 验证北向流入
- 机构卖出 → 注意 thesis 是否变化

**坑**：游资买入不代表基本面好，更多是情绪传染。

## 8. 大宗交易

**为什么重要**：大资金的"暗箱操作"窗口。

**怎么查**：
- 沪深交易所每日"大宗交易"披露
- 同花顺 / Wind 都有专项页面

**解读规则**：
- 折价大宗（如 -8%）→ 卖方急于出货，警惕
- 平价 / 溢价大宗 → 接盘方有充分理由（信心信号）
- 接盘方为"机构专用席位"→ 机构在低位建仓（强信号）
- 高位连续折价大宗 → 老股东在跑路

## 9. 股东户数

**为什么重要**：散户筹码 vs 机构筹码的平衡指标。

**怎么查**：
- 半年报 / 季报披露
- 同花顺 / Wind 都有专项页面

**解读规则**：
- 户数环比减少 + 股价没怎么动 = 机构在悄悄收筹码（强信号）
- 户数环比增加 + 股价上涨 = 散户在追，警惕高位接盘
- 户数环比减少 + 股价下跌 = 散户割肉、机构吸筹（极强信号）

## 10. 公司公告

最被低估的信号源。**每次重要公告必读**。

**关注公告类型**：
- 大客户订单 / 框架协议（chokepoint 验证）
- 定增 / 配股 / 可转债（融资用途要看清楚，扩产 = 好，补流 = 警惕）
- 股权激励（行权条件揭示管理层信心水位）
- 限售股解禁（解禁前 1 个月可能有抛压）
- 大股东 / 高管增减持（高管买入是强信号；减持要看比例和原因）
- 股权质押率变化（> 60% 黄灯，> 80% 红灯）
- 实控人 / 董事长变更（高度敏感）
- 重大资产重组 / 收购
- 立案调查 / 监管问询（直接红线）

**怎么查**：
- 上市公司公告：巨潮资讯网（cninfo.com.cn）官方源
- 聚合：同花顺、东方财富、Wind 的"公告"板块
- 重大事件预警：选股软件的"自选股公告推送"

## 怎么把信号源拼成 thesis

一个**满分 chokepoint** 候选应该同时满足：

| 维度 | 理想信号 |
|------|---------|
| 产业链位置 | 上游单点 + 全球 ≤ 3 家能做 / 国产替代独苗 |
| 客户资本开支 | 主要下游云厂商 / 晶圆厂资本开支同比 +20% 以上 |
| 政策催化 | 出口管制升级 / 大基金入股 / 国务院产业规划点名 |
| 北向资金 | 近 3 个月净流入 + 持股比例上行 |
| 公募基金 | 持仓家数环比增加（最新季报）|
| 龙虎榜 | 近期出现机构席位买入 |
| 公司公告 | 大客户订单 / 高管增持 / 扩产 |
| 财务健康 | 净利润正、经营性现金流匹配、商誉低、质押率低 |
| 估值 | 不要求绝对低估，但要相对合理（PE / PS 不离同业过远）|

**满 7-8 项**：A 类候选，进入深度尽调
**满 5-6 项**：B 类候选，观察
**< 5 项**：候选不成立，继续找
